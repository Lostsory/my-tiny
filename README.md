# My-tiny
[tinypng](https://tinypng.com/)配合git的一个命令行工具
## Installation

Install through npm:
```
npm install -g my-tiny
tiny c -key <your tinyPng's appkey>
```

## Usage
```
git add .
tiny s
git commit -m <msg>
```
